# analise-dados-bi
Dashboard de Performance Financeira (Bootcamp Engenharia de Dados). Análise estratégica de R$ 118,73M em vendas, com foco em lucratividade por segmento e país (destaque para França com 22,38% do lucro). Desenvolvido no Power BI com processos de ETL via Power Query para limpeza e modelagem de dados brutos da Sample Financial
